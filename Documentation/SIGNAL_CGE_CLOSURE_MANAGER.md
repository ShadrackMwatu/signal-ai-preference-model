# Signal CGE Closure Manager

The closure manager stores and validates macro and market-clearing assumptions.

## Supported Closure Families

- base closure
- static comparative closure
- recursive dynamic closure
- savings-driven investment
- investment-driven savings
- fixed government savings
- flexible direct tax replacement
- fixed foreign savings
- flexible exchange rate
- factor full employment
- unemployment or fixed wage option

Closure validation currently checks known closure names and missing closure variables. Full closure testing will be expanded when nonlinear equations and solver residuals are active.
