# Tests

Tests cover CSV loading, model training/persistence, FastAPI endpoints, and
research-grade CGE/SAM export outputs. Module 1 tests cover deterministic
synthetic data ingestion and validation. Market intelligence tests cover
feature extraction, model training, prediction outputs, clustering, adaptive
retraining, drift detection, model-version logs, NLP dissatisfaction outputs,
opportunity generation, and privacy safeguards.
They use Python's built-in `unittest` runner.
