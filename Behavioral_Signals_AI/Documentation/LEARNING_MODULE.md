# Learning Module

Last updated: 2026-05-08

## Purpose

The Learning module gives users plain-language explanations of Signal concepts, SML, CGE models, policy simulation, demand scoring, and related topics.

## Main File

`learning/ai_teaching/explainer.py`

## Current Concepts

- Signal
- Revealed Preference
- Behavioral Signals
- Demand Classification
- Opportunity Scoring
- Unmet Demand
- Emerging Trends
- SAMs
- CGE Models
- SML
- Policy Simulation

## Dashboard Integration

The Gradio Learning tab provides:

- topic dropdown
- explanation textbox
- learning dashboard refresh
- recent lessons
- recurring issues
- recommended fixes
- apply, ignore, and rollback actions

## Future Direction

The learning module should evolve into an AI tutor for the platform, explaining not just concepts but specific model results, validation errors, and scenario outputs.
