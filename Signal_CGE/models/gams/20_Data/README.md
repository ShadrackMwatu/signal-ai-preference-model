\# Local GAMS Data Files



Large SAM and GAMS data files are intentionally excluded from Hugging Face deployment.



For full local GAMS execution, place the following files in this folder:



\- KEN\_SAM\_2020.xlsx

\- experiment\_KENv17.xlsx



The Hugging Face app should still run using fallback, sample, or demo data when these local files are not available.s
